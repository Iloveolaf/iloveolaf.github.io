108-1 學期回顧

發表於 2020-02-04 更新於 2021-02-17 分類於 NTU 閱讀次數：
文章字數： 4.3k 所需閱讀時間 ≈ 4 分鐘

一早 工廠實習(1)

這門課應該可以榮登大一上最有趣的課排名前三名(?)，雖然我一開始上課超級不習慣，因為圍裙的繩子每次都會勒到脖子，導致上課到一半就吸不到氣、想吐之類的，不過後來就好很多。我們這組學工法的順序是磨工 -> 铣工 -> 鑽工 -> 鉗工 -> 焊工 -> 車工，超級逆序(?)，幾乎所有要加工的工件都是師傅先給我們而不是我們自己做的 xD 比較印象深刻的大概是我在每一個工法都把機器或器具用壞…（師傅我很抱歉 QQ ），再來就是在焊工的時候做的雪寶跟最後做出來的風車吧，真的成就感滿滿。

雪寶一開始其實想要做小隻一點，結果因為鐵絲太短的話很難弄彎，還有如果手的長度剪太短根本焊不上去，於是它就變得比我的臉還要大了(?)，不過成品看起來蠻不錯的，而且師傅居然認得出來是雪寶 xD 太酷了。

至於風車的話就是用整個學期做的工件組成的，然後風車競賽的時候原本想要四個人（我、莊皓宇、梁晏慈、施柏均）一起上去丟臉(因為我們四個人的不知道為什麼都不會轉 xD )，結果我們一起放到風力集中處後就…剩晏慈的不會轉了 xD 抱歉背叛妳 xDD 最後就去噴漆上色啦~原本想噴銀色的但是銀色沒了 QAQ 只好噴裸色，不過裸色也很好看(只是我沒噴好有些地方沒噴到 QQ

二早 靜力學(2) 劉建豪

沒錯，我這學期一下午沒課。

原本學期初預計的學習模式是，上課聽 ppt，回家看原文書跟寫完後面的習題，但習題有夠多，每一章都差不多有 100 題左右，根本寫不完 QQ 所以後來就只寫教授給的習題了。一開始真的超不習慣英語授課，雖然東西都聽得懂但翻成英文就整個不知道在幹嘛，但我覺得後來有越來越習慣了。（不過剛剛看了下學期的課表，為什麼必帶只要是英語授課的就會是我的班阿 QQ ）總之我覺得學到蠻多東西的，而且教授真的很棒，都會很有耐心地回答問題，小考也通常不會太難，期中考們就不一定（就像我的第二次期中考爛到谷底 QQ ）。

二午 機械工程概論(0)

老實說我覺得還蠻無聊的…就在講一些概括性的東西，不過畢竟是概論嘛，也沒辦法強求太多(?)

二午 程式設計(3) 孔令傑

在 2019 的 SITCON 就聽過教授的演講覺得超棒，再加上學分不夠就去加簽了，反正我的 C++ 基礎也不是太好(X)，所以基本上這堂課還蠻甜涼的(?)，每個禮拜只要花半小時寫掉作業，稍微看看影片裡有沒有不會的觀念就好。上課的話就去教室聽教授回答同學的問題，然後一邊寫掉上課的 exercise，寫完就回家了 ww

印象比較深刻的大概是期中期末的 exam 跟 project。先講 exam 好了，考試時間是 3 個小時(嗎)，有一個很酷的規定是居然可以上網查資料耶，酷爆 ww 然後不能跟活體生物(助教以外)溝通 ww，真的是很可愛的規定 xD，不過兩場考試我差不多 1 小時寫完就走了，所以這就是為什麼我不太清楚到底考試時間有多久(X 其實兩場考試都只有 4 題啦，但是因為是後測，所以雖然都是水題我還是都檢查了大概 30 分鐘這樣，所以實際 coding 時間大概是 30 分鐘(?)，然後就順利的小考跟大考都 100 分啦~

再來就是 project，恩，期中的 project 真的是慘不忍睹，我的組員真的很雷，再加上期中 project 只有兩個禮拜的時間，而在這兩個禮拜裡我有四個期中考跟兩份期中報告要做，所以整個忙到不行，導致就算知道是 dp 也來不及寫。期末的 project 我就換組了，換到一個很可愛的組 ww 組員人都超讚的而且超級凱瑞，我們的主題是遊戲，他們幫我處理好所有的圖片、書面報告之類的，真的超級感謝他們。我主要是負責 coding 的部分，碰了蠻多以前沒有碰過的東西，也算是點了一點點技能(?)。總之，其實我真的覺得教授教得很好（我還是有聽課！），是一堂真的很推薦的課，不過新手可能要努力一點就是了，聽說這堂課對新手的 loading 其實蠻重的。

三早 普通物理學實驗(1) 助教課

就做實驗阿，稍微值得一提的是用通靈做出來的腔體共振實驗(我直接看腔體猜共振頻率 ww )、有夠有趣的口頭報告(柯南物理分析)、還有怪到爆的期末考(?)剩下好像就沒什麼好說的了 ww（喔還有啦，我覺得兩個助教都很可愛 ww ）

三午/五早 微積分甲上(4) 周謀鴻

免修考睡過頭的概念，不過再重新複習一次也好，畢竟我微積分也不是太熟，不過有點像是誤上賊船就是了(X 第一次小考就直接出 hyperbolic function 的相關證明，助教你讀數學研究所也不代表我們以後要念好嗎(X 不過後來感覺就變一種常態了(?)，助教幾乎每次都要跟我們講一次「喔好像不小心出太難了，可是最難的那題不是我出的喔」xD 反正習慣就好。不過學期中各種出事，導致我的微積分成績整個大爆炸，不過還好期末考很水（沒記錯大概 1 小時就寫完了）然後沒提到正課是因為我很少去上(X，除非我自己來不及複習之類的才會去上課，不然整天被噹就很煩 QQ（還好不算出席率，剛剛稍微算了一下，我這學期出席不到1/5(X)）下學期還在考慮要不要換教授。（不過我好喜歡我們班的助教喔，他很可愛而且教得很好 ww ）（好啦其實我也覺得教授講還不錯，可是就不習慣不能一邊聽微積分一邊用手機做其他事，蠻常被誤會是在玩手機的）

四午/五早 普通物理學甲上(4) 趙治宇

跟著治宇快樂學物理(?)其實真的還蠻快樂而且學到蠻多東西的 ww 真心覺得教授教的很棒，而且觀念什麼的都講很清楚，還常常自己示範物理現象(X 剛剛想了想整個學期上下來印象最深刻的居然是教授說 Jack 跟 Rose （鐵達尼號）是笨死的，到底什麼鬼(X 而且上課很自由，真的很自由，基本上不吵教授都沒事，所以可以一邊學自己的東西一邊上物理，真的超棒。最後畢竟普物是必帶，所以下學期沒有意外的話就還是同一個教授摟 ww

四午 鋼琴作品與演奏欣賞(2) 張欽全

莫名其妙在初選第二階段選上的(?)，不過也不錯啦~不過有種被騙的感覺，就是我以為真的會有演奏欣賞，結果根本超少 QQ 反正蠻少去上課的，不過有幾次認真聽了課，真心不太能同意教授的各種想法(?)所以其實我沒有很喜歡這門課，恩。

五午 健康體適能(1/0) 王翊澤

莫名其妙我就自願當小老師了(?)，其實我真的忘記為什麼我要自願了 ww 一開始以為小老師會很忙，不過後來大家都不借器材，整學期似乎沒做到什麼事 ww 我覺得我們體育課還蠻酷的(?) 可以免費請三次假、期中考考帶操 + 設計遊戲、還有期末考就是體適能測驗再測一次 ww 而且如果大隊接力有進決賽（多比一天）的話還可以多一次假，超讚 ww 啊既然有四次假當然就請好請滿啊(X 印象中請假都是因為準備數學之美期中跟生理期吧，剩下時間我還是有乖乖去上課的 xD 至於期中期末 ww 期中我們組真的有夠天才，躲避球跟籃球混合真的超好笑，沒聲音也玩的超開心 ww 期末的話，進步就是 10 分，平 5 分，退步 0 分，但反正老師說女生第一次立定跳有超過 180 就算退步也不扣分，就隨便亂跳啦(X（是說期末的坐姿體前彎成功達到 41 公分耶好開心，高中都沒有超過 10 公分的 ww ）總之就是一堂好玩的體育課啦 xD

五午 數學之美(2) 呂學一

這門課是我這學期最喜歡的一堂課。雖然前一、兩堂(?)有一點點無聊，不過後面超有趣（講質數、有理數、實數），而且真的完全沒有用到微積分欸，酷爆。喔然後教授人也超好，有什麼問題都可以問，如果教授不確定的話也會跟我們一起找答案，真的很喜歡這種老師。整個學期上下來一整個收穫滿滿，學到超多超棒的東西~（雖然我是有聽好幾個人說整個上不下去啦 xD 不過我很喜歡數學所以(O)）是說教授感覺很跟得上時代欸 ww 上課真的是各種梗，還會放同學在投票區的留言到 ppt 上（不過下載的時候就沒有了，個人認為是讓ppt很精湛的一部份，應該要放的(X）說到 ppt 很精湛的部分，教授的動畫真的是超猛，到底怎麼做到的好想知道(?)。總之如果是喜歡數學的人，真的大推這堂課，內容超級充實，阿考題就不多說，背各種人物細節什麼的真的是很不擅長，真的要說就是翻轉我 GPA 的一堂課(X

至於下學期的話，扣掉免修的普化丙(3)，剩下的必修就有 14 學分惹，所以大概只需要再多選一堂課(?)，希望我選的到 QAQ

小後記：其實感覺上大學跟高中好像沒有差很多，除了上課可以拿手機查資料（雖然高中也可以(X）、不用每節課都要去上、課表可以自己排以外好像沒有太大差別。
