Windows 跟 iOS/iPadOS 互傳檔案方法

發表於 2020-07-30 更新於 2020-07-31 分類於 其他 閱讀次數：
文章字數： 1.4k 所需閱讀時間 ≈ 1 分鐘

前(ㄈㄟˋ)言(ㄏㄨㄚˋ)

因為用 iPad Air 加上 notability (一個很棒的筆記程式)，很常需要用到 iOS 內建的「檔案」 app 把 pdf 之類的檔案匯入 notability，但我 pdf 習慣在電腦上下載，可是 windows 又沒辦法直接把檔案傳到iOS 內建的「檔案」 app 裡面，所以參考了一點資料來完成這件事。(參考網站放在下面)

步驟

步驟 1

在 Windows 上選擇要分享的資料夾(裡面有沒有東西都可以)，右鍵按內容

步驟 2 設定權限

2-1

「共用」 -> 「進階共用」 -> 勾選「共用此資料夾」 -> 「權限」。

2-2 (可跳過)

接下來就是設定權限，可以直接使用 Everyone 來操作，也可以新增使用者。如果不想新增使用者就可以跳過。

「新增」 -> 「進階」 -> 「立即尋找」。在搜尋結果裡面找到自己電腦的名字就可以了，點選之後按確定。

2-3

回到設定權限的地方，點選剛剛新增的使用者或者 Everyone，勾選允許「完全控制」、「變更」，接著一路按確定回到內容。

2-4

點選「共用」，在下拉選單中找到剛剛新增的使用者或者 Everyone，新增完成後點選下面剛剛新增的人員，權限層級改成「讀取/寫入」，接著按右下角的「共用」 -> 「完成」 -> 「關閉」。

2-5 (可跳過)

接下來是要設定 Windows 的開機密碼，如果有設定過的話可以跳過。

點左下角 windows 圖案，點選使用者圖案，選取「變更帳戶設定」。

「登入選項」 -> 「密碼」 -> 「變更」

步驟 3 取得電腦IP

「win 鍵 + R」 -> 輸入「cmd」 -> 輸入「ipconfig」，應該會長的像下圖，找到「無線區域網路介面卡 Wi-Fi 」中的「IPv4 位址」。

步驟 4 連線

打開 iPhone 或 iPad 的「檔案」 app，點選右上角的「…」。

點選「連接伺服器」。(注意這時候 iPhone 或 iPad 用的網路跟電腦要是一樣的)

輸入「smb:// + 剛剛記下來的 IP」。ex. smb://10.1.1.17

名稱就輸入電腦帳戶名稱，密碼則是電腦密碼。

這樣就結束了，以後就可以讓 iPhone 或 iPad 跟電腦雙向傳檔案了~
