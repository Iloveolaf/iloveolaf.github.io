CF 1359C Mixing Water

發表於 2020-08-09 分類於 CF 閱讀次數：
文章字數： 1.2k 所需閱讀時間 ≈ 1 分鐘

題目

題目連結：Link

題目敘述：給熱水、冷水的溫度 $h$ 和 $c$，問輪流倒熱水冷水(至少倒入一杯熱水)，需要多少杯才能讓溫度最接近 $t$。($c \le t \le h$)

難易度：1700(應該不到吧，是不是給錯了)

解法

首先可以發現只有兩種倒法，$n$ 杯熱水 + $n$ 杯冷水 或 $n + 1$ 杯熱水 + $n$ 杯冷水，在 $n$ 一樣的時候，後者的溫度一定會比前者高，又前者的溫度一定是 $\frac{h + c}{2}$，所以 $t \le \frac{h + c}{2}$ 的答案一定是 $2$。

接下來就考慮 $t \gt \frac{h + c}{2}$ 的狀況，這裡又可以再分成 $n + 1$ 杯熱水 + $n$ 杯冷水 和 $n + 2$ 杯熱水 + $n + 1$ 杯冷水的情況，看哪一個比較接近 $t$ 就好了 ~ $n$ 的估計就用 $\frac{(n + 1) * h + n * c}{2n + 1} = t$ 這條式子。這裡為了避免浮點數誤差的問題所以用整數運算。

$$\lvert \frac{(n + 1) \times h + n \times c}{2n + 1} - t \rvert \lt \lvert \frac{(n + 2) \times h + (n + 1) \times c}{2n + 3} - t \rvert$$

可以變成

$$\lvert (n + 1) \times h + n \times c - t \times (2n + 1) \rvert \times (2n + 3) \lt \lvert (n + 2) \times h + (n + 1) \times c - t \times (2n + 3) \rvert \times (2n + 1) $$
